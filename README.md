4. **A/B Testing & Statistical Analysis:**  
   - Conducted hypothesis testing for conversion rate differences.

5. **Dashboarding:** Designed Power BI dashboard for ROI and conversion insights.  
6. **Performance Evaluation:** Compared campaign groups and measured uplift metrics.
